## 편집 팁

### CapCut 사용 시
1. 프로젝트 생성 (1080p, 30fps)
2. 녹화 영상 임포트
3. 텍스트 추가 (Templates → Minimal 스타일)
4. 트랜지션: "Fade" 또는 "Slide" (0.3초)
5. 음악 추가 + 볼륨 30-40%
6. Export: 1080p, High Quality

### 텍스트 스타일 통일
```
폰트: Inter (없으면 Roboto)
제목: Bold, 48-64px
본문: Medium, 24-32px
색상: 흰색 (#FFFFFF) 또는 검정 (#000000)
그림자: 약간 (가독성)
```
