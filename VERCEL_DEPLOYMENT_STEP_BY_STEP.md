## ⚠️ 만약 에러가 발생하면

### 문제 1: "Build Failed" 또는 "Error" 메시지

**해결 방법:**

1. **"Build Logs" 섹션 클릭**
   - 화면 아래쪽에 있습니다
   - "> Build Logs" 클릭

2. **에러 메시지 확인**
   - 빨간색 에러 메시지를 찾으세요
   - 에러 내용을 복사하세요

3. **에러 내용을 알려주세요**
   - 저에게 에러 메시지를 보여주시면 해결해드립니다

---

### 문제 2: "Deployment Settings 1 Recommendation" 경고

**해결 방법:**

1. **"> Deployment Settings 1 Recommendation" 클릭**
   - 화면에 보이는 이 섹션을 클릭하세요

2. **추천 사항 확인**
   - 어떤 설정이 필요한지 확인하세요
   - 대부분 환경 변수 관련입니다

3. **설정 적용**
   - 추천 사항을 따라 설정하세요
   - 또는 저에게 알려주시면 안내해드립니다

## 🎯 다음 단계 (배포 완료 후)

### 4단계: 환경 변수 확인하기

**배포가 완료된 후:**

1. **화면 왼쪽 메뉴에서 "Settings" 클릭**
   - 프로젝트 이름 아래에 있습니다

2. **"Environment Variables" 클릭**
   - 왼쪽 메뉴에서 선택

3. **5개 환경 변수가 모두 있는지 확인:**
   - ✅ `NEXT_PUBLIC_SUPABASE_URL`
   - ✅ `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - ✅ `SUPABASE_SERVICE_ROLE_KEY`
   - ✅ `ENCRYPTION_KEY`
   - ✅ `NEXT_PUBLIC_PYTHON_SERVER_URL`

4. **만약 누락된 변수가 있다면:**
   - "Add New" 버튼 클릭
   - 변수 이름과 값 입력
   - Production, Preview, Development 모두 체크
   - "Save" 버튼 클릭
   - **중요:** 저장 후 "Redeploy" 버튼 클릭!

---

### 5단계: 도메인 연결하기 (선택사항)

**`fieldnine.io` 도메인을 연결하려면:**

1. **화면 왼쪽 메뉴에서 "Settings" 클릭**

2. **"Domains" 클릭**
   - 왼쪽 메뉴에서 선택

3. **"Add Domain" 버튼 클릭**

4. **도메인 입력:**
   ```
   fieldnine.io
   ```

5. **DNS 설정 안내 따르기**
   - Vercel이 DNS 설정 방법을 알려줍니다
   - 도메인 제공업체에서 설정하면 됩니다
