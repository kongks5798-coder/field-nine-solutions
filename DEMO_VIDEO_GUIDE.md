# 데모 영상 제작 가이드

## 🎬 X(Twitter)용 데모 영상 제작 방법

### 1. 화면 녹화 도구
- **Windows**: Xbox Game Bar (Win + G) 또는 OBS Studio
- **Mac**: QuickTime Player 또는 ScreenFlow
- **온라인**: Loom, ScreenRec

### 2. 녹화 시나리오 (30-60초)

#### 시퀀스 1: 첫 인상 (5초)
- 웜 아이보리 배경의 미니멀 대시보드
- "Field Nine AI" 타이틀
- "오늘 예상 절약: 52,000원" 카드

#### 시퀀스 2: 추천 상품 카드 (10초)
- 3개의 추천 상품 카드 스크롤
- 각 카드의 가격, 절약 금액 강조
- 데이터 소스 배지 표시

#### 시퀀스 3: AI 채팅 인터랙션 (15초)
- 채팅 입력창에 "운동화 추천해줘" 입력
- "보내기" 버튼 클릭
- AI 응답 표시: "이 옷 추천해요! 예상 가격 45,000원 (20% 세일 예정)"

#### 시퀀스 4: CTA (5초)
- "지금 시작하기" 버튼
- URL 표시: fieldnine.io

### 3. 편집 팁

#### 필수 요소
- **자막 추가**: 주요 기능 설명 (한국어)
- **배경 음악**: 미니멀한 인스트루멘탈 (저작권 무료)
- **속도 조절**: 중요 부분은 0.8x, 빠른 부분은 1.2x
- **텍스트 강조**: "52,000원 절약", "AI 추천" 등

#### 권장 도구
- **무료**: Canva, CapCut, DaVinci Resolve
- **유료**: Final Cut Pro, Adobe Premiere

### 5. 최적화 팁

- **비율**: 16:9 또는 9:16 (모바일 최적화)
- **길이**: 30-60초 (X 알고리즘 최적)
- **첫 3초**: 강렬한 훅 (절약 금액 강조)
- **CTA**: 마지막 5초에 명확한 액션 버튼
