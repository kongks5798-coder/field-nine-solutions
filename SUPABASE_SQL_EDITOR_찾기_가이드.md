# 🔍 Supabase SQL Editor 찾기 완벽 가이드

## 🚨 문제: SQL Editor가 안 보여요

SQL Editor를 찾는 여러 방법을 알려드립니다!

---

## 🎯 방법 1: 왼쪽 메뉴에서 찾기

### 1-1. Supabase 프로젝트 대시보드 확인
1. Supabase.com에 접속되어 있는지 확인
2. **"field-nine-solutions"** 프로젝트가 선택되어 있는지 확인
3. 프로젝트 대시보드 화면이 보여야 합니다

### 1-2. 왼쪽 메뉴 전체 확인
왼쪽에 세로로 나열된 메뉴를 **아래로** 스크롤하면서 다음을 찾아보세요:

**일반적으로 보이는 메뉴들:**
- Table Editor
- **SQL Editor** ← 이걸 찾으세요!
- Authentication
- Storage
- Edge Functions
- Database
- API
- Settings

### 1-3. SQL Editor 찾기
1. 왼쪽 메뉴를 **아래로** 끝까지 스크롤
2. **"SQL Editor"** 또는 **"SQL 편집기"** 찾기
   - 아이콘: 데이터베이스 아이콘 또는 SQL 관련 아이콘
   - 텍스트: **"SQL Editor"**
3. **"SQL Editor"** 클릭

---

## 🎯 방법 2: 검색 기능 사용

### 2-1. 검색창 찾기
1. 화면 상단에 검색창이 있습니다
   - 보통 오른쪽 상단에 위치
   - "Q" 아이콘 또는 "Search" 텍스트
2. 검색창 클릭

### 2-2. 검색하기
1. 검색창에 다음을 입력: `SQL Editor`
2. **Enter** 키 누르기
3. 검색 결과에서 **"SQL Editor"** 클릭

---

## 🎯 방법 3: URL로 직접 접속

### 3-1. 현재 URL 확인
브라우저 주소창을 확인하세요:
- 현재 URL이 보입니다
- 예: `https://supabase.com/dashboard/project/vhpefsmwpoixzEpu`

### 3-2. URL 수정
1. 주소창의 마지막 부분을 다음으로 변경:
   ```
   /sql/new
   ```
2. 전체 URL 예시:
   ```
   https://supabase.com/dashboard/project/vhpefsmwpoixzEpu/sql/new
   ```
3. **Enter** 키 누르기

---

## 🎯 방법 4: Database 메뉴에서 찾기

### 4-1. Database 메뉴 찾기
1. 왼쪽 메뉴에서 **"Database"** 찾기
2. **"Database"** 클릭

### 4-2. SQL Editor 찾기
1. Database 페이지가 열립니다
2. 상단 탭 또는 메뉴에서 **"SQL Editor"** 찾기
3. **"SQL Editor"** 클릭

---

## 🎯 방법 5: 프로젝트 설정에서 찾기

### 5-1. Settings 메뉴 확인
1. 왼쪽 메뉴에서 **"Settings"** 찾기
2. **"Settings"** 클릭

### 5-2. 하위 메뉴 확인
Settings를 클릭하면 왼쪽에 더 많은 메뉴가 나타납니다:
- General
- API
- Database
- **SQL Editor** ← 여기에 있을 수도 있습니다!

---

## 🎯 방법 6: 화면 새로고침

### 6-1. 페이지 새로고침
1. **F5** 키 누르기
   - 또는 **Ctrl + R** 키
2. 페이지가 새로고침됩니다
3. 왼쪽 메뉴를 다시 확인

### 6-2. 브라우저 캐시 지우기
1. **Ctrl + Shift + Delete** 키 누르기
2. **"캐시된 이미지 및 파일"** 체크
3. **"데이터 삭제"** 클릭
4. Supabase를 다시 접속

---

## 🎯 방법 7: 다른 프로젝트 확인

### 7-1. 프로젝트가 맞는지 확인
1. 화면 상단에 프로젝트 이름이 보입니다
2. **"field-nine-solutions"**가 맞는지 확인
3. 다른 프로젝트를 선택했다면 올바른 프로젝트 선택

### 7-2. 프로젝트 목록에서 확인
1. 화면 상단 왼쪽에 프로젝트 이름 옆에 화살표가 있을 수 있습니다
2. 클릭하면 프로젝트 목록이 나타납니다
3. **"field-nine-solutions"** 선택

---

## 🎯 방법 8: 직접 URL 입력 (가장 확실한 방법)

### 8-1. 프로젝트 ID 확인
1. Supabase 대시보드에서
2. Settings → General 클릭
3. **"Project ID"** 확인
   - 예: `vhpefsmwpoixzEpu`
4. 이 ID를 복사

### 8-2. SQL Editor URL 만들기
1. 브라우저 주소창에 다음 URL 입력:
   ```
   https://supabase.com/dashboard/project/[프로젝트ID]/sql/new
   ```
2. `[프로젝트ID]`를 실제 프로젝트 ID로 바꾸기
   - 예: `https://supabase.com/dashboard/project/vhpefsmwpoixzEpu/sql/new`
3. **Enter** 키 누르기

---

- [ ] Database 메뉴에서 SQL Editor 찾기

## 💡 팁

1. **메뉴가 접혀있을 수 있어요**
   - 왼쪽 메뉴를 아래로 끝까지 스크롤해보세요
   - 화면이 작으면 메뉴가 접혀있을 수 있습니다

2. **브라우저 확대/축소 확인**
   - 화면이 너무 확대되어 있으면 메뉴가 안 보일 수 있습니다
   - **Ctrl + 0** 키로 기본 크기로 되돌리기

3. **다른 브라우저로 시도**
   - Chrome에서 안 보이면 Edge나 Firefox로 시도해보세요

4. **직접 URL 사용**
   - 가장 확실한 방법은 직접 URL을 입력하는 것입니다

## 🚨 여전히 안 보이면

### 문제 해결:
1. **Supabase에 로그인되어 있는지 확인**
2. **올바른 프로젝트를 선택했는지 확인**
3. **브라우저를 완전히 닫고 다시 열기**
4. **시크릿 모드에서 시도해보기**

---

## 🎯 가장 빠른 방법 (권장)

**직접 URL로 접속:**

1. Supabase 대시보드에서 프로젝트 ID 확인
   - Settings → General → Project ID
2. 브라우저 주소창에 입력:
   ```
   https://supabase.com/dashboard/project/[프로젝트ID]/sql/new
   ```
3. Enter 키 누르기

이렇게 하면 SQL Editor가 바로 열립니다!
