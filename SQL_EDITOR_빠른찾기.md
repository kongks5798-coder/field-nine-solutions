# ⚡ SQL Editor 빠르게 찾기

## 🎯 가장 빠른 방법: 직접 URL로 접속

### 1단계: 프로젝트 ID 확인
1. Supabase 대시보드에서
2. **Settings** → **General** 클릭
3. **"Project ID"** 확인
   - 예: `vhpefsmwpoixzEpu`
4. 이 ID를 복사

### 2단계: SQL Editor URL 만들기
1. 브라우저 주소창에 다음 입력:
   ```
   https://supabase.com/dashboard/project/vhpefsmwpoixzEpu/sql/new
   ```
   - `vhpefsmwpoixzEpu`를 실제 프로젝트 ID로 바꾸세요
2. **Enter** 키 누르기

✅ **SQL Editor가 바로 열립니다!**

---

## 🔍 다른 방법들

### 방법 1: 왼쪽 메뉴에서 찾기
1. 왼쪽 메뉴를 **아래로** 끝까지 스크롤
2. **"SQL Editor"** 찾기
3. 클릭

### 방법 2: 검색 사용
1. 화면 상단 검색창 클릭
2. `SQL Editor` 입력
3. Enter 키
4. 검색 결과에서 클릭

### 방법 3: Database 메뉴
1. 왼쪽 메뉴에서 **"Database"** 클릭
2. 상단 탭에서 **"SQL Editor"** 찾기
3. 클릭

---

## 💡 팁

- **메뉴가 안 보이면:** 왼쪽 메뉴를 아래로 스크롤
- **가장 확실한 방법:** 직접 URL 입력
- **프로젝트 ID 확인:** Settings → General → Project ID
