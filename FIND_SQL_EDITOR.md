# 🔍 Supabase SQL Editor 찾기 완벽 가이드

## 🚨 문제: SQL Editor가 안 보여요

SQL Editor를 찾는 여러 방법을 알려드립니다!

---

## ⚡ 가장 빠른 방법: 직접 URL로 접속

### 1단계: 프로젝트 ID 확인
1. Supabase 대시보드에서
2. **Settings** → **General** 클릭
3. **"Project ID"** 확인
   - 예: `vhpefsmwpoixzEpu`
4. 이 ID를 복사

### 2단계: SQL Editor URL 만들기
1. 브라우저 주소창에 다음 입력:
   ```
   https://supabase.com/dashboard/project/vhpefsmwpoixzEpu/sql/new
   ```
   - `vhpefsmwpoixzEpu`를 실제 프로젝트 ID로 바꾸세요
2. **Enter** 키 누르기

✅ **SQL Editor가 바로 열립니다!**

---

## 🎯 방법 1: 왼쪽 메뉴에서 찾기

### 1-1. 왼쪽 메뉴 전체 확인
1. 화면 왼쪽에 세로 메뉴가 있습니다
2. 메뉴를 **아래로** 끝까지 스크롤합니다
3. 다음 메뉴들을 찾아보세요:
   - Table Editor
   - **SQL Editor** ← 이걸 찾으세요!
   - Authentication
   - Storage
   - Edge Functions
   - Database
   - API
   - Settings

### 1-2. SQL Editor 클릭
1. **"SQL Editor"** 또는 **"SQL 편집기"** 찾기
2. 클릭

---

## 🎯 방법 2: 검색 기능 사용

### 2-1. 검색창 찾기
1. 화면 상단 오른쪽에 검색창이 있습니다
   - "Q" 아이콘 또는 "Search" 텍스트
2. 검색창 클릭

### 2-2. 검색하기
1. 검색창에 입력: `SQL Editor`
2. **Enter** 키 누르기
3. 검색 결과에서 **"SQL Editor"** 클릭

---

## 🎯 방법 3: Database 메뉴에서 찾기

### 3-1. Database 메뉴 클릭
1. 왼쪽 메뉴에서 **"Database"** 찾기
2. **"Database"** 클릭

### 3-2. SQL Editor 탭 찾기
1. Database 페이지가 열립니다
2. 상단 탭에서 **"SQL Editor"** 찾기
3. **"SQL Editor"** 클릭

---

## 🎯 방법 4: URL 수정하기

### 4-1. 현재 URL 확인
브라우저 주소창을 확인:
- 예: `https://supabase.com/dashboard/project/vhpefsmwpoixzEpu`

### 4-2. URL 끝에 추가
1. 주소창의 마지막에 다음을 추가:
   ```
   /sql/new
   ```
2. 전체 URL 예시:
   ```
   https://supabase.com/dashboard/project/vhpefsmwpoixzEpu/sql/new
   ```
3. **Enter** 키 누르기

---

## 🎯 방법 5: 페이지 새로고침

### 5-1. 새로고침하기
1. **F5** 키 누르기
   - 또는 **Ctrl + R** 키
2. 페이지가 새로고침됩니다
3. 왼쪽 메뉴를 다시 확인

### 5-2. 브라우저 캐시 지우기
1. **Ctrl + Shift + Delete** 키 누르기
2. **"캐시된 이미지 및 파일"** 체크
3. **"데이터 삭제"** 클릭
4. Supabase를 다시 접속

---

## 📋 체크리스트

다음 방법들을 순서대로 시도해보세요:

- [ ] **가장 빠른 방법:** 직접 URL로 접속 (`/sql/new` 추가)
- [ ] 왼쪽 메뉴를 아래로 스크롤해서 SQL Editor 찾기
- [ ] 검색창에서 "SQL Editor" 검색
- [ ] Database 메뉴에서 SQL Editor 찾기
- [ ] 페이지 새로고침 (F5)
- [ ] 올바른 프로젝트 선택했는지 확인

## 💡 팁

1. **메뉴가 접혀있을 수 있어요**
   - 왼쪽 메뉴를 아래로 끝까지 스크롤해보세요

2. **화면 크기 확인**
   - 화면이 너무 확대되어 있으면 메뉴가 안 보일 수 있습니다
   - **Ctrl + 0** 키로 기본 크기로 되돌리기

3. **가장 확실한 방법**
   - 직접 URL을 입력하는 것이 가장 빠르고 확실합니다
