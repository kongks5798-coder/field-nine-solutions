# Field Nine 사용자 가이드

## 시작하기

### 1. 회원가입

1. 메인 페이지에서 **"지금 시작하기"** 클릭
2. 로그인 페이지에서 **"회원가입"** 클릭
3. 이메일과 비밀번호 입력
   - 비밀번호: 8자 이상, 대소문자, 숫자, 특수문자 포함
4. **"회원가입"** 버튼 클릭

### 2. 로그인

#### 이메일/비밀번호 로그인
1. 이메일과 비밀번호 입력
2. **"로그인"** 버튼 클릭

#### 소셜 로그인
- **Google**: "Google 계정으로 계속하기" 클릭
- **Kakao**: "Kakao로 3초 만에 시작하기" 클릭

> **참고**: 소셜 로그인은 관리자가 OAuth를 설정해야 사용할 수 있습니다.

## 대시보드

### 홈
- 통계 카드: 총 주문 수, 처리 중, 총 매출, 활성 사용자
- 빠른 액션: 주문 동기화, 분석, 설정
- 최근 활동: 최근 이벤트 목록

### 주문 동기화
1. **"주문 동기화"** 버튼 클릭
2. Python 서버가 실행 중이어야 합니다
3. 동기화된 주문이 목록에 표시됩니다

#### 주문 관리
- **상태 변경**: 상태 배지 클릭 → 새 상태 선택
- **삭제**: 삭제 버튼 클릭 → 확인

#### 검색 및 필터
- 검색: 주문번호, 고객명, 이메일로 검색
- 필터: 상태별 필터링

### 분석 및 통계
- 기간 선택: 7일, 30일, 90일, 전체
- 주요 지표: 총 매출, 총 주문 수, 평균 주문 금액
- 상태별 분포: 주문 상태별 통계
- 일별 매출 Top 5: 일별 매출 순위

### 설정
- **프로필 설정**: 이메일, 이름 변경
- **알림 설정**: 이메일, 푸시, SMS 알림 설정
- **보안 설정**: 비밀번호 변경, 2FA (향후 지원)

## 문제 해결

### 로그인이 안 될 때
1. 이메일/비밀번호 확인
2. 이메일 인증 확인 (필요 시)
3. 브라우저 쿠키 확인

### 주문 동기화가 안 될 때
1. Python 서버 실행 확인 (`http://127.0.0.1:8000`)
2. 네트워크 연결 확인
3. 서버 로그 확인

### 소셜 로그인이 안 될 때
1. OAuth 설정 확인 (관리자)
2. 브라우저 팝업 차단 확인
3. 리다이렉트 URL 확인
