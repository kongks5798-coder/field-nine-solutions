## ❌ 부족한 핵심 기능 (상용화 필수)

### 1. 재고 자동 차감 ❌ (치명적)
**문제**: 주문 생성 시 재고가 자동으로 차감되지 않음
**영향**: 재고 관리 불가능, 중복 주문 가능
**필요**: 주문 생성 시 트리거로 재고 차감

### 2. 주문 상태 자동 전환 ❌ (중요)
**문제**: 배송 시작 시 자동으로 SHIPPED로 전환되지 않음
**필요**: 송장번호 입력 시 자동 상태 전환

### 3. 수익 계산 완성 ❌ (중요)
**문제**: 수수료 계산 로직 미완성
**필요**: 플랫폼 수수료, 결제 수수료 자동 계산

### 4. 상품 상세 페이지 ❌ (중요)
**문제**: 상품 상세 보기 페이지 없음
**필요**: `/products/[id]` 페이지

### 5. 분석 대시보드 완성 ❌ (중요)
**문제**: 기본 구조만 있고 실제 차트/인사이트 부족
**필요**: 매출 추이, 상품별 분석, 트렌드 분석

### 6. PWA 최적화 ⚠️ (중요)
**문제**: 기본 manifest만 있고 Service Worker 없음
**필요**: 오프라인 지원, 푸시 알림, 앱 설치

### 7. 에러 핸들링 강화 ⚠️ (중요)
**문제**: 일부 페이지에서 에러 처리 미흡
**필요**: 전역 에러 핸들러, 사용자 친화적 메시지

### 8. 성능 최적화 ⚠️ (중요)
**문제**: 이미지 최적화, 로딩 상태 개선 필요
**필요**: Next.js Image 최적화, Suspense 경계
