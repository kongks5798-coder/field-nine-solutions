### 6.2 연동 인터페이스

```yaml
# KEPCO 연동 API 사양
kepco_interface:
  protocol: REST/MQTT
  authentication: OAuth 2.0 + mTLS
  data_format: IEC 61850 / CIM
  update_interval: 15분

  endpoints:
    - /api/v1/meter/reading      # 계량 데이터
    - /api/v1/settlement/request # 정산 요청
    - /api/v1/grid/status        # 계통 상태
    - /api/v1/price/forecast     # 가격 예측

# 전력거래소 연동 API 사양
kpx_interface:
  protocol: WebSocket
  authentication: PKI Certificate
  data_format: OASIS Energy Market XML

  subscriptions:
    - 실시간 SMP
    - 보조서비스 가격
    - 계통 주파수
```
