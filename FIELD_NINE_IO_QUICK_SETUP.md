# ⚡ fieldnine.io 빠른 설정 가이드

## 🎯 3단계로 완료하기

### 1️⃣ Vercel에 도메인 추가
1. Vercel 대시보드 → 프로젝트 → Settings → Domains
2. "Add Domain" → `fieldnine.io` 입력
3. DNS 설정 안내 확인

### 2️⃣ DNS 설정
1. 도메인 관리 페이지 접속
2. Vercel이 제공하는 DNS 레코드 추가
3. DNS 전파 대기 (몇 분)

### 3️⃣ 환경변수 업데이트 (선택사항)
- 자동 감지 로직이 있으므로 환경변수는 선택사항
- 또는 `NEXT_PUBLIC_ARBITRAGE_API_URL`을 `https://fieldnine.io`로 설정
