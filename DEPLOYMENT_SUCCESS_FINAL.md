## 📋 즉시 확인해야 할 사항

### 1. 배포 URL 접속 (1분)

#### 메인 페이지
```
https://your-app.vercel.app
```
- 브라우저에서 접속
- 페이지가 정상적으로 로드되는지 확인

#### 차익거래 페이지
```
https://your-app.vercel.app/arbitrage
```
- 차익거래 대시보드 접속
- 대시보드가 정상적으로 표시되는지 확인

---

### 2. 브라우저 개발자 도구 확인 (2분)

#### Console 확인
1. **F12 키 누르기**
   - 개발자 도구 열기

2. **Console 탭 확인**
   - 에러 메시지가 없는지 확인

3. **환경변수 확인**
   ```javascript
   console.log(process.env.NEXT_PUBLIC_ARBITRAGE_API_URL)
   ```
   - 설정한 URL이 출력되는지 확인

#### Network 확인
1. **Network 탭 클릭**
2. **페이지 새로고침 (F5)**
3. **API 요청 확인**
   - API 요청이 정상적으로 전송되는지 확인
   - 200 상태 코드인지 확인

---

### 3. API 엔드포인트 테스트 (2분)

#### 헬스 체크
```
https://your-app.vercel.app/api/health
```
- JSON 응답 확인

#### 기회 조회
```
https://your-app.vercel.app/api/opportunities
```

## 🚨 문제 해결

### 문제 1: 페이지가 로드되지 않음
**해결**: 브라우저 캐시 삭제 (Ctrl+Shift+Delete)

### 문제 2: API 연결 오류
**해결**: 환경변수 값 확인 및 Redeploy

### 문제 3: WebSocket 연결 실패
**해결**: WebSocket URL 및 환경변수 확인
