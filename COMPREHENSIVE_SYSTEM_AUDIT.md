### 1. 터미널 및 개발 환경 문제 (5개) - 🔴 Critical

#### 문제 1.1: Node 프로세스 중복 실행
- **증상**: `dev.lock` 파일 충돌로 개발 서버 시작 불가
- **영향**: 개발 작업 불가능
- **점수 감점**: -100점
- **해결 방법**: 
  - 실행 중인 모든 node 프로세스 종료
  - `.next/dev.lock` 파일 삭제
  - 프로세스 관리 스크립트 추가

#### 문제 1.2: 빌드 경고
- **증상**: Middleware 파일 convention deprecated 경고
- **영향**: 미래 호환성 문제
- **점수 감점**: -50점
- **해결 방법**: `proxy`로 마이그레이션

#### 문제 1.3: 환경 변수 검증 부족
- **증상**: 런타임 에러 가능성
- **영향**: 프로덕션 안정성 저하
- **점수 감점**: -50점

#### 문제 1.4: 에러 로깅 시스템 부재
- **증상**: 에러 추적 불가능
- **영향**: 디버깅 어려움
- **점수 감점**: -50점

#### 문제 1.5: 개발 도구 설정 부족
- **증상**: 디버깅 도구 없음
- **영향**: 개발 효율성 저하
- **점수 감점**: -50점
