### 옵션 1: 간단한 지시 (권장)

```
Cursor야, Field Nine 공식 홈페이지 (app/page.tsx)를 Tesla 2026 Edition 스타일로 완전히 리디자인해줘.

요구사항:
1. 디자인:
   - 미래지향적이고 고급스러운 Tesla 스타일
   - 다크 그라데이션 배경 (#0A0A0A → #1A1A1A → #0F0F0F)
   - 웜 아이보리 텍스트 (#F5F5F0)
   - Cyan/Blue 액센트 (#06B6D4, #2563EB)
   - 넓은 여백 (white space 충분히)
   - 부드러운 애니메이션 (Framer Motion)
   - 3D 효과와 호버 인터랙션

2. 섹션 구성:
   - Hero: 대형 비주얼 + 강력한 헤드라인 + CTA
   - Problem: 고객의 문제점 명확히 제시
   - Solution: Field Nine 솔루션 상세 설명 (6-8개 기능)
   - Features: 각 기능별 상세 설명 + 아이콘 + 통계
   - Benefits: 구체적인 이점 (비용 절감, 시간 절약, 수익 증가)
   - Proof: 고객 사례, 통계, 로고 (social proof)
   - Pricing: 간단한 가격 안내
   - CTA: 최종 행동 유도

3. 기능 설명:
   - 재고 자동 관리: AI 예측, 자동 차감, 재고 알림
   - 주문 자동 처리: 멀티채널 통합, 자동 동기화, 상태 관리
   - 수익 최적화: 동적 가격, 수수료 계산, 수익 분석
   - 비용 절감: RTX 5090 로컬 AI, 98% 클라우드 비용 절감
   - 보안: End-to-end 암호화, RLS 정책, 99.9% 가동률
   - 분석 대시보드: 실시간 통계, 차트, 리포트
   - AI 자동화: 수요 예측, 가격 최적화, 트렌드 분석
   - 통합: 쿠팡, 11번가, 네이버쇼핑, 스마트스토어 등

4. 신뢰감 요소:
   - 구체적인 통계 (98% 비용 절감, 99.9% 가동률, 0.1초 응답)
   - 고객 로고 (가능하면)
   - 사용자 후기
   - 보안 인증 배지
   - 실시간 모니터링 표시

5. 기술 요구사항:
   - 반응형 디자인 (모바일 완벽 지원)
   - 접근성 (ARIA 레이블, 키보드 네비게이션)
   - SEO 최적화
   - 빠른 로딩 속도
   - 부드러운 스크롤 애니메이션

코드 변경 없이 설명만 추가하지 말고, 실제로 디자인을 완전히 개선해줘.
```

### 옵션 2: 상세한 지시

```
Cursor야, Field Nine 공식 홈페이지를 Tesla처럼 미래지향적이고 신뢰감 있는 고급스러운 디자인으로 완전히 리디자인해줘.

디자인 원칙:
1. Tesla 2026 스타일:
   - Neo-minimalism: 불필요한 요소 제거, 핵심만 강조
   - Brutalism elements: 대담한 타이포그래피, 강한 대비
   - Organic flow: 자연스러운 애니메이션 흐름
   - Saturation revival: Cyan/Blue 액센트로 생동감
   - Clean geometry: 깔끔한 기하학적 형태

2. 색상 팔레트:
   - 배경: 다크 그라데이션 (#0A0A0A → #1A1A1A → #0F0F0F)
   - 텍스트: 웜 아이보리 (#F5F5F0)
   - 액센트: Cyan (#06B6D4), Blue (#2563EB)
   - 보조: 회색 톤 (#9CA3AF)

3. 타이포그래피:
   - 헤드라인: text-6xl ~ text-8xl, font-light (Tesla 스타일)
   - 서브헤드라인: text-2xl ~ text-4xl, font-medium
   - 본문: text-lg ~ text-xl, font-normal
   - 넓은 줄간격 (leading-relaxed)

섹션별 요구사항:

1. Hero Section:
   - 전체 화면 높이 (min-h-screen)
   - 대형 헤드라인: "당신의 비즈니스를 AI로 혁신"
   - 서브헤드라인: 구체적인 가치 제안
   - 배경: 미래 도시 스카이라인 (그라데이션 + 기하학적 형태)
   - 파티클 효과 (부드러운 애니메이션)
   - CTA 버튼: 큰 버튼, 프레스 애니메이션
   - 통계 4개: 일일 처리 주문, 비용 절감, 가동률, 응답 시간

2. Problem Section:
   - 고객의 문제점 3-4개 제시
   - 각 문제마다 아이콘 + 설명
   - 다크 배경, Cyan 액센트

3. Solution Section:
   - Field Nine 솔루션 소개
   - "How it works" 시각적 설명
   - 단계별 프로세스 (1-2-3)

4. Features Section (핵심):
   - 6-8개 주요 기능 상세 설명
   - 각 기능별:
     * 큰 아이콘 (3D 효과)
     * 제목 + 서브타이틀
     * 상세 설명 (3-4줄)
     * 구체적인 통계/수치
     * Bullet points (3-4개)
   - 카드 레이아웃: 겹치는 카드, 호버 시 확대
   - 스크롤 애니메이션: 각 카드가 순차적으로 나타남

5. Benefits Section:
   - 구체적인 이점:
     * 비용 98% 절감 (구체적 금액 예시)
     * 시간 90% 절약 (구체적 시간)
     * 수익 23% 증가 (평균)
   - Before/After 비교 (선택)
   - 통계 차트/그래프

6. Proof Section:
   - 고객 로고 (가능하면)
   - 사용자 후기 (3-4개)
   - 통계 슬라이더
   - 실시간 모니터링 표시

7. Pricing Section:
   - 간단한 가격 안내
   - 베타 오퍼 강조
   - CTA 버튼

8. Final CTA:
   - 큰 배경
   - 강력한 메시지
   - 여러 CTA 버튼

기능 설명 상세화:

1. 재고 자동 관리:
   - AI 기반 수요 예측
   - 자동 재고 차감/복구
   - 저재고 알림
   - 멀티웨어하우스 관리
   - 통계: 90% 시간 절약, 0% 오류율

2. 주문 자동 처리:
   - 멀티채널 통합 (쿠팡, 11번가, 네이버쇼핑, 스마트스토어)
   - 자동 주문 동기화
   - 상태 자동 업데이트
   - 송장번호 자동 입력
   - 통계: 100% 자동화, 0.1초 처리

3. 수익 최적화:
   - 동적 가격 조정
   - 수수료 자동 계산
   - 수익 분석 대시보드
   - 통계: 평균 23% 수익 증가

4. 비용 절감:
   - RTX 5090 로컬 AI
   - 클라우드 비용 98% 절감
   - 자동 스케일링
   - 통계: 월 $10,000 → $200

5. 보안:
   - End-to-end 암호화
   - RLS 정책
   - 자동 백업
   - 통계: 99.9% 가동률

6. 분석 대시보드:
   - 실시간 통계
   - 차트 및 그래프
   - 커스텀 리포트
   - 통계: 실시간 업데이트

7. AI 자동화:
   - 수요 예측 (98% 정확도)
   - 가격 최적화
   - 트렌드 분석
   - 통계: 98% 정확도

8. 통합:
   - 쿠팡, 11번가, 네이버쇼핑, 스마트스토어
   - API 연동
   - 웹훅 지원

애니메이션:
- Framer Motion 사용
- 스크롤 기반 애니메이션
- 호버 효과 (스케일, 그림자)
- 부드러운 전환 (ease-in-out)
- 파티클 효과

반응형:
- 모바일: 세로 레이아웃, 큰 터치 영역
- 태블릿: 2열 그리드
- 데스크톱: 3-4열 그리드

접근성:
- ARIA 레이블
- 키보드 네비게이션
- 스크린 리더 지원
- 충분한 색상 대비
```
