# ⚡ Vercel 환경변수 추가 - 빠른 단계

## 🎯 지금 바로 따라하기

### 현재 상태
- ✅ Vercel Environment Variables 페이지 열림
- ✅ "Add New" 섹션 보임

---

## 🚀 7단계로 완료하기

### 1️⃣ Key 입력
```
NEXT_PUBLIC_ARBITRAGE_API_URL
```
- Key 필드에 정확히 입력

### 2️⃣ 배포 URL 확인
- Deployments 탭 → 최신 배포 → Domains 확인
- 예: `https://field-nine-solutions.vercel.app`

### 3️⃣ Value 입력
```
https://field-nine-solutions.vercel.app
```
- Value 필드에 실제 URL 입력

### 4️⃣ Environments 선택
- "All Environments" 선택

### 5️⃣ Save 클릭
- 하단 오른쪽 Save 버튼

### 6️⃣ 목록 확인
- 환경변수가 목록에 추가됨

### 7️⃣ Redeploy
- Deployments → 최신 배포 → ... → Redeploy

---
