# ⚡ NEXT_PUBLIC_ARBITRAGE_API_URL 추가 - 간단 버전

## 🎯 3분 안에 완료하기

### 현재 상태
- ❌ `NEXT_PUBLIC_ARBITRAGE_API_URL`이 목록에 없음
- ✅ Environment Variables 페이지 열려있음

---

## 🚀 5단계로 추가하기

### 1️⃣ Key 입력
```
NEXT_PUBLIC_ARBITRAGE_API_URL
```
- "Add New" 섹션의 Key 필드에 입력

### 2️⃣ 배포 URL 확인
- Deployments 탭 → 최신 배포 → Domains 확인
- 예: `https://field-nine-solutions.vercel.app`

### 3️⃣ Value 입력
```
https://field-nine-solutions.vercel.app
```
- Value 필드에 실제 URL 입력

### 4️⃣ Save 클릭
- 하단 오른쪽 Save 버튼

### 5️⃣ Redeploy
- Deployments → 최신 배포 → ... → Redeploy

---

## ✅ 완료!

**보스, 이제 목록에 보일 겁니다!** 🚀
