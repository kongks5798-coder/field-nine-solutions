# 📝 Supabase 스키마 실행 단계별 가이드

## 🎯 현재 위치
- ✅ Supabase 프로젝트 대시보드에 있음
- ✅ 다음 단계: SQL Editor에서 스키마 실행

---

## 🎬 1단계: SQL Editor 열기

### 1-1. 왼쪽 메뉴 확인
1. 화면 왼쪽에 세로 메뉴가 있습니다
2. 여러 메뉴 아이콘과 텍스트가 보입니다

### 1-2. SQL Editor 찾기
1. 왼쪽 메뉴에서 **"SQL Editor"** 찾기
   - 아이콘: 데이터베이스 또는 SQL 관련 아이콘
   - 텍스트: **"SQL Editor"** 또는 **"SQL 편집기"**
2. **"SQL Editor"** 클릭

✅ **확인:** SQL Editor 페이지가 열리면 성공!

---

## 🎬 2단계: 새 쿼리 만들기

### 2-1. "New query" 버튼 찾기
1. SQL Editor 화면 상단 또는 왼쪽 상단 확인
2. **"+ New query"** 또는 **"새 쿼리"** 버튼 찾기
   - 버튼 색상: 초록색 또는 파란색
   - 아이콘: **"+"** 기호
3. **"+ New query"** 버튼 클릭

✅ **확인:** 새로운 빈 쿼리 창이 열리면 성공!

---

## 🎬 3단계: 첫 번째 스키마 파일 열기

### 3-1. 파일 탐색기 열기
1. **Windows 키 + E** 키 누르기
   - 또는 작업 표시줄에서 폴더 아이콘 클릭

### 3-2. 프로젝트 폴더로 이동
1. 주소창에 다음 경로 입력:
   ```
   C:\Users\polor\field-nine-solutions
   ```
2. **Enter** 키 누르기

### 3-3. supabase 폴더 열기
1. **"supabase"** 폴더 찾기
2. **"supabase"** 폴더 더블클릭

### 3-4. schema.sql 파일 열기
1. **"schema.sql"** 파일 찾기
2. **"schema.sql"** 파일 더블클릭
   - 메모장 또는 VS Code로 열립니다

✅ **확인:** schema.sql 파일 내용이 보이면 성공!

---

## 🎬 4단계: 파일 내용 복사하기

### 4-1. 전체 선택
1. 파일이 열려있는 편집기에서
2. **Ctrl + A** 키 누르기
   - 파일 내용 전체가 파란색으로 선택됩니다

### 4-2. 복사하기
1. **Ctrl + C** 키 누르기
   - 내용이 클립보드에 복사됩니다

✅ **확인:** 복사 완료!

---

## 🎬 5단계: SQL Editor에 붙여넣기

### 5-1. Supabase 브라우저 탭으로 돌아가기
1. 브라우저에서 Supabase SQL Editor 탭 클릭

### 5-2. 텍스트 영역 클릭
1. SQL Editor 가운데 큰 텍스트 입력 영역 클릭
2. 커서가 깜빡이기 시작하면 준비 완료

### 5-3. 붙여넣기
1. **Ctrl + V** 키 누르기
2. ✅ SQL 코드가 나타나면 성공!

---

## 🎬 6단계: 스키마 실행하기

### 6-1. "Run" 버튼 찾기
1. 화면 상단 또는 하단 확인
2. **"Run"** 또는 **"실행"** 버튼 찾기
   - 보통 초록색 또는 파란색 버튼
   - 오른쪽 상단 또는 하단에 위치

### 6-2. 실행하기
1. **"Run"** 버튼 클릭
2. ⏳ 1-2초 대기

### 6-3. 결과 확인
1. 화면 오른쪽 또는 아래에 결과 표시
2. ✅ **"Success"** 또는 **"성공"** 메시지 확인

✅ **첫 번째 스키마 완료!**

---

## 🎬 7단계: 두 번째 스키마 실행하기

### 7-1. 새 쿼리 만들기
1. **"+ New query"** 버튼 다시 클릭

### 7-2. 두 번째 파일 열기
1. 파일 탐색기에서
2. `supabase` 폴더 안에서
3. **"schema_subscriptions.sql"** 파일 더블클릭

### 7-3. 복사하기
1. **Ctrl + A** (전체 선택)
2. **Ctrl + C** (복사)

### 7-4. 붙여넣기
1. Supabase SQL Editor로 돌아가기
2. 새 쿼리 창의 텍스트 영역 클릭
3. **Ctrl + V** (붙여넣기)

### 7-5. 실행하기
1. **"Run"** 버튼 클릭
2. ✅ 성공 메시지 확인

✅ **두 번째 스키마 완료!**
